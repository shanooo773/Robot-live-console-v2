## 👋 Welcome to AnyBot

Welcome to **AnyBot** — your **remote robotics lab in the browser**.

With AnyBot, you can:

- 🤖 Program real robots
- 🧠 Learn ROS2 step-by-step
- 🌍 Access everything from anywhere
- 🧪 Run experiments without local setup

## 📅 Step 2: Book a Session (Reserve Your Robot)

Before you start coding, you need to **book a robot session** on AnyBot.

This ensures you get **dedicated access to the robot/simulation environment**.

### 🔄 Booking Flow Overview

```
Start New Session → Select Robot → Select Date → Select Time → Confirm
```

### 🟢 1. Click “Start New Session”

On your dashboard:

- Locate the **"Start New Session"** button
- Click to begin booking

### 🤖 2. Select Robot

You will see available robots.

For this course:

👉 Select **Hiwonder Robot**

### 📆 3. Select Date

- Choose a date from the calendar
- Available dates will be enabled

### ⏰ 4. Select Time Slot

- Pick an available time slot
- Each slot gives you access for a fixed duration

### ✅ 5. Confirm Booking

- Review your selection
- Click **Confirm / Book Session**

![](../assets/booking_workflow.png)

Now go **Back to dashboard**, and click **Projects**

![](/home/muneeb/Desktop/HiWonder-MentorPi_A1-Learning_Paths/assets/session_booked.png)

![](../assets/go_to_projects.png)

## 🎯 What Happens After Booking?

After your session is booked, you will see your upcoming sessions at bottom of Projects page.

- Once your session time starts, the **Enter Console** option will be enabled
- Robot becomes available
- You can access the robot through Robot Development console
- Click the **Enter Console** button, you will enter into **Robot Development Console (RDC)**

![](../assets/upcoming_sessions.png)

## 👋 Robot Development Console (RDC)

Now you are inside the Robot Development Console (RDC), this is your development environment and we will refer to this as RDC in further guides and tutorials.

The RDC will initially look as follows:

![](../assets/rdc_initial_view.png)

You can increase the size of windows in RDC as follows

![](../assets/rdc_annotated.png)

Next,

- Open the File Editor / Viewer
- Open the Terminal / Command Interface

![](../assets/rdc_code_flow.png)

You have the following windows / Panes in Code Editor. The Code Editor Window is divided into multiple panes, you can use each pane for different purpose.

![](/home/muneeb/Desktop/HiWonder-MentorPi_A1-Learning_Paths/assets/rdc_panes.png)

- Code Editor: Used to editing the code from code file
- Terminal: Used to execute commands.
- File Viewer: Used to select and open code files.

## 🎯 What's Up Next?

Now you are inside the **Robot Development Console**,  your live workspace connected to the robot.

This means:

- 🤖 You now have **active access to the robot/simulation**
- 💻 You can **write, run, and test ROS2 code**
- 📡 Your code will directly interact with the robot in real-time
